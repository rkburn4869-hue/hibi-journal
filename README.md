# Hibi — 朝夕10分のジャーナリング

朝と夜、それぞれ約10分でできる簡潔なジャーナリング（日記）の PWA です。
世界中のジャーナリング手法（Five Minute Journal / Morning Pages / 感謝ジャーナル / CBTジャーナリング など）から、効果が出やすい要素を抽出して設計しています。

## 機能

- 🌅 **朝のジャーナリング** … 感謝 / 今日の意図 / 自分への一言 / ブレインダンプ
- 🌙 **夜のジャーナリング** … 良かったこと / 学び / 明日への一手 / 気分スコア
- 📈 気分の推移グラフ（直近14日）
- 🔥 連続記録（ストリーク）と累計記録
- 🔔 朝・夜のリマインド通知（ブラウザ通知）
- ✏️ プロンプト（質問）の追加・編集・並べ替え
- 📱 PWA対応（ホーム画面に追加・オフライン起動）
- 💾 データは端末のブラウザ内（localStorage）にのみ保存

## 使い方

`index.html` を開くだけで動きます。
ただし PWA（通知・オフライン）は `https://` または `localhost` 経由で開いた場合のみ有効です。

ローカルで確認する場合:

```bash
python3 -m http.server 8000
# ブラウザで http://localhost:8000/ を開く
```

## 技術

- HTML / CSS / バニラ JavaScript（外部ライブラリなし）
- Service Worker + Web App Manifest による PWA 化
